# 株式会社ミタックス コーポレートサイト

「水と空気を整え、暮らしと産業を支える。」をコンセプトにした、
信頼感・技術力・清潔感・採用力を伝える企業サイトです。
HTML / CSS / JavaScript のみで動作する静的サイトのため、特別な環境構築は不要です。

---

## 1. ファイル構成

```
mitax-website/
├── index.html            … TOPページ（全セクションを1ページに集約）
├── recruit.html          … 採用情報ページ（独立ページ）
├── css/
│   └── style.css         … デザイン全般（色・余白・レイアウト）
├── js/
│   └── main.js           … スクロール演出・数字カウントアップ等
├── assets/
│   ├── images/           … 写真・ロゴ（★ここを差し替える）
│   └── gen-placeholders.js … プレースホルダー画像の生成用（任意）
└── README.md             … このファイル
```

`index.html` には以下の10セクションが含まれます。
1. TOP（ヒーロー）／ 2. 会社案内 ／ 3. 事業内容 ／ 4. 施工実績 ／
5. 選ばれる理由 ／ 6. 数字で見るミタックス ／ 7. 社員インタビュー ／
8. 採用情報（→ recruit.html へ誘導）／ 9. 代表メッセージ ／ 10. お問い合わせ

## 3. 写真の差し替え方法 ★最重要

写真はすべて `assets/images/` 内の **SVGプレースホルダー** になっています。
**同じファイル名で実写真を置く**だけで反映されます（拡張子を変える場合は下記参照）。

| ファイル名 | 使われている場所 | 推奨サイズ |
|---|---|---|
| `hero.svg` | TOP メインビジュアル | 1920×1080px 以上 |
| `about.svg` | 会社案内の写真 | 1200×1000px |
| `biz-1〜5.svg` | 事業内容の各カード | 1200×825px |
| `work-1〜6.svg` | 施工実績の各カード | 1200×900px |
| `person-1〜4.svg` | 社員インタビューの写真（縦） | 800×1000px |
| `message.svg` | 代表メッセージの写真（縦） | 900×1100px |
| `recruit-hero.svg` | 採用ページ メインビジュアル | 1920×1080px 以上 |
| `recruit-life.svg` | TOPの採用誘導バンド背景 | 1200×900px |
| `cta.svg` | 予備のCTA背景 | 1920×800px |
| `favicon.svg` | ブラウザタブのアイコン | 正方形 |

### 差し替えの手順（例：TOPのメイン写真）
- **方法A（おすすめ）**：写真を `hero.svg` という名前で上書き保存（SVG以外でもファイル名を合わせればOK）
- **方法B**：`hero.jpg` などを `assets/images/` に置き、HTML側の
  `<img src="assets/images/hero.svg">` を `hero.jpg` に書き換え

> JPG/PNG をそのまま使う場合は、HTMLの該当 `<img src="...">` の拡張子を合わせてください。
> HTML内の写真位置には `<!-- ▼写真差し替え：… -->` のコメントを添えています。

## 4. 文章の差し替え方法

文章は HTML に直接書かれています。テキストエディタ（メモ帳でも可）で
`index.html` / `recruit.html` を開き、該当する日本語を書き換えてください。

特に差し替えが必要な「仮の情報」：
- **会社概要**（`index.html` 内「会社案内」の表）… 代表者名・所在地・許認可番号・資本金など
- **代表者名**（`〇〇 〇〇` となっている箇所）
- **電話番号 / メールアドレス**（`000-000-0000` / `info@example.co.jp`）
- **住所**（`〒000-0000 〇〇県…`）
- **施工実績**のタイトル・年・エリア（`〇〇ビル…` など）
- **数字で見る働く環境**（採用ページ）… サンプル数値のため実数値へ
- **募集要項**（採用ページ）… 給与・勤務地などの条件

> `〇〇` や `000` は仮の値です。本番公開前に必ず実際の情報へ置き換えてください。

## 5. 色を変えたいとき

`css/style.css` の先頭にある `:root { … }` の変数を変更すると、
サイト全体の色が一括で変わります。

```css
--navy-800: #0a2540;   /* メインのネイビー */
--blue-500: #2a93d5;   /* アクセントの水色 */
--paper:    #f6f9fc;   /* オフホワイト背景 */
```

## 6. お問い合わせフォームについて

フォームは現在**デモ動作**（送信すると確認メッセージが出るのみ）です。
実際にメールを受信するには、以下のいずれかの設定が必要です。

- フォーム送信サービス（Formspree / Google フォーム / SSGフォーム 等）に連携
- サーバー側（PHP等）で送信処理を実装

`index.html` の `<form data-demo-form …>` 周辺と、`js/main.js` の
「7. 簡易フォーム送信」コメント部分を参照してください。

## 7. 公開（アップロード）方法

`mitax-website` フォルダの中身を、そのままレンタルサーバーや
ホスティング（さくら／エックスサーバー／Netlify／Vercel 等）にアップロードするだけで公開できます。
ビルド作業は不要です。

## 8. 動作環境・対応

- スマートフォン／タブレット／PC に完全対応（レスポンシブ）
- 最新版の Chrome / Edge / Safari / Firefox で動作確認を推奨
- スクロールアニメーション・数字カウントアップは JavaScript で動作します
  （OSの「視差効果を減らす」設定時は、アニメーションを自動的に控えめにします）
