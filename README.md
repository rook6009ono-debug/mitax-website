# 株式会社ミタックス コーポレートサイト

「水と空気を整え、暮らしと産業を支える。」をコンセプトにした、
信頼感・技術力・清潔感・採用力を伝える企業サイトです。
HTML / CSS / JavaScript のみで動作する静的サイトのため、特別な環境構築は不要です。

> 会社概要・連絡先・事業内容・採用条件などは、現行サイト（mitax-apr.co.jp）と
> 公開求人情報をもとに**実在情報を反映**しています。最新の内容に合わせて適宜更新してください。

## 1. ページ構成（多ページ構成）

| ファイル | ページ | 主な内容 |
|---|---|---|
| `index.html` | TOP | ヒーロー、各ページへの導線、選ばれる理由、数字、施工実績・採用の抜粋 |

| `recruit.html` | 採用情報 | 未経験歓迎、資格取得支援、働く環境の数字、先輩社員の声、1日の流れ、募集職種、募集要項、福利厚生、選考フロー |

## 2. 閲覧・確認方法

- かんたん確認：`index.html` をブラウザにドラッグ＆ドロップ
- 推奨（フォント等を正しく確認する場合）：簡易サーバーで開く

```bash
python -m http.server 8000      # → http://localhost:8000
npx serve .                      # Node.js の場合
```

## 3. 写真の差し替え方法 ★最重要

写真はすべて `assets/images/` 内の **SVGプレースホルダー** です。
**同じファイル名で実写真を上書き**するだけで全ページに反映されます。

| ファイル名 | 使用箇所 | 推奨サイズ |
|---|---|---|
| `hero.svg` | TOP メインビジュアル | 1920×1080px 以上 |
| `recruit-hero.svg` | 採用ページ メイン | 1920×1080px 以上 |
| `about.svg` | 会社案内ヒーロー／TOP会社紹介 | 1200×1000px |
| `message.svg` | 代表メッセージ（縦長） | 900×1100px |
| `biz-1〜5.svg` | 事業内容の各事業 | 1200×825px |
| `work-1〜6.svg` | 施工実績の各事例 | 1200×900px |
| `person-1〜4.svg` | 先輩社員の声（縦） | 800×1000px |
| `recruit-life.svg` | 採用バンド背景 | 1200×900px |
| `cta.svg` | CTA帯の背景 | 1920×800px |
| `logo.png` | ロゴ（ヘッダー・フッター） | 透過PNG |
| `favicon.svg` | ブラウザタブのアイコン | 正方形 |

> JPG/PNG を使う場合は、各HTMLの `<img src="...">` の拡張子を合わせてください。
> 写真位置には `<!-- ▼写真差し替え… -->` のコメントを添えています。

## 4. 文章・情報の差し替え

文章は各HTMLに直接記述しています。テキストエディタで開いて書き換えてください。
現時点では以下を実在情報で反映済みです（変更があれば更新してください）。

- 会社名・所在地・電話/FAX・設立・資本金・代表者・従業員数・売上高・取引銀行（`company.html`）
- 事業内容（`business.html`）／採用条件（`recruit.html`）

なお、以下は今後の追記・確定をおすすめします。
- **建設業許可・指定給水装置工事事業者などの許認可番号**（現行サイトに記載がないため未掲載）
- **沿革の詳細な年表**（主要な節目のみ記載）
- **施工実績の実写真・実案件名**（現在は代表的な施工例）
- **代表者・社員の顔写真**

## 5. 色を変えたいとき

`css/style.css` 先頭の `:root { … }` 変数を変更すると全体に反映されます。

```css
--navy-800: #0a2540;   /* メインのネイビー */
--blue-500: #2a93d5;   /* アクセントの水色 */
```

---

## 6. お問い合わせフォームについて

フォームは現在**デモ動作**（送信で確認メッセージのみ）です。
実際にメールを受信するには、フォーム送信サービス（Formspree 等）連携か、
サーバー側（PHP等）での送信処理の実装が必要です。
`contact.html` の `<form data-demo-form>` 周辺と `js/main.js`「7. 簡易フォーム送信」を参照してください。

---

## 7. 地図（アクセス）について

会社案内・お問い合わせページの地図は、所在地（国分寺市東戸倉1-15-16）の
Googleマップ埋め込みです。住所を変更する場合は、各HTMLの `<iframe src="...">` 内の
`q=` パラメータの住所を書き換えてください。
